Banking data (including SWIFT-codes/BIC) for Germany, Austria, Switzerland.

## Installation

Add this line to your application's Gemfile:

    gem 'banking_data'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install banking_data
